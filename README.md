# 🛡️ Insurance CRM

A modern, AI-powered CRM for an insurance agent — clients, households, policies,
renewals, claims, WhatsApp, and a Claude-powered AI layer. Runs locally on your PC.

## Quick start
```bash
npm install     # install dependencies (one-time)
npm run setup   # create the database + load demo data (one-time)
npm run dev     # run the app
```
Then open **http://localhost:3000**.

See [PROJECT.md](./PROJECT.md) for the plan, phases, and decisions.
